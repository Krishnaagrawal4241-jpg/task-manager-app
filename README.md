# Task Manager App

## About the Project
This is a simple full-stack Task Manager application that I built using React and Flask. The main idea of this project is to understand how frontend and backend communicate with each other.

In this app, users can add tasks, view them, mark them as completed, and delete them.

## Tech Stack Used
- Frontend: React (Vite)
- Backend: Flask (Python)
- Database: SQLite
- ORM: SQLAlchemy

## How it Works
The React frontend sends requests to the Flask backend using APIs. The backend processes the request and stores or retrieves data from the SQLite database.

## Features
- Add a new task  
- View all tasks  
- Mark task as completed  
- Delete a task  

## How to Run the Project

### Backend
cd backend  
python app.py  

### Frontend
cd frontend/vite-project  
npm install  
npm run dev
